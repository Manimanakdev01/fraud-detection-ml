# Fraud Detection using Machine Learning

## 📌 Problem Statement
Detect fraudulent transactions (binary classification) using transactional and risk-based features.

## 🧠 Approach
- Data cleaning & preprocessing
- Feature scaling & leakage prevention
- Model selection (Logistic Regression / SVM)
- Calibration & threshold tuning
- Evaluation using PR, ROC-AUC, confusion matrix
- Production-aware considerations

## 📊 Evaluation Metrics
- Precision
- Recall
- F1-score
- ROC-AUC
- Calibration curve

## ⚠️ Production Challenges Considered
- Data Drift
- Feature Drift
- Concept Drift
- Class Imbalance
- Training–Serving Skew

## 🛠️ Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- Matplotlib / Seaborn

## 🚀 Future Improvements
- Online monitoring
- Drift detection
- Model retraining strategy
